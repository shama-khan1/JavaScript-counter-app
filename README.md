JavaScript Counter App

A simple and responsive Counter App built using HTML, CSS, and JavaScript.
This project demonstrates basic DOM manipulation, event handling, and responsive design.

🚀 Live Demo

🔗 https://shama-khan1.github.io/javaScript-counter-app/

📌 Features

- Simple and clean UI
- Button click counter functionality
- Responsive design using CSS media queries
- Beginner-friendly JavaScript logic

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

⚙️ How It Works

When the user clicks the Count button, a JavaScript function runs and increases the number displayed on the screen.
